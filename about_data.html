FIFA22 Dataset

Data for this assignment(Lab1) is obtained from Kaggle. It contains all the data about all the fifa 22 player's attributes and information.


Attributes:

Name: Players Name (categorical variable)
Age: Player's Age
Height: Player's Height in cms
Weight: Player's Weight in kgs
Overall: Player's Overall Rating in FIFA 22
Potential: Player's Potential Rating in FIFA 22
BestPosition: Player's Best Position in FIFA 22 (categorical variable)
Club: Player's homeclub (categorical variable)
Nationality: Nationality of players (categorical variable)
ValueEUR: Player's Transfer Market Value in EURO
WageEUR: Player's Weekly Wage in EURO
PreferredFoot: Player's preferred foot - Left/Right (categorical variable)
SkillMoves: Player's SKill Moves (5 star, 4 star, 3 star, 2 star, 1 star) Rating (categorical variable)
PaceTotal: Speed at which the player can run
ShootingTotal: Shooting attribute score
PassingTotal: Passing attribute score
DribblingTotal: Dribbling attribute score
DefendingTotal: Defending attribute score
Crossing: Crossing attribute score
Finishing: Finishing attribute score
Jumping: Jumping attribute score
Stamina: Player's Stamina
Vision: Player's Vision Capability
BallControl: Player's Ball Control Capability
FKAccuracy: Free kick accuracy
Penalties: Player's ability to score Penalties
